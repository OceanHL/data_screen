# 电商平台数据可视化实时监控系统

## 项目特点

- 电商平台数据可视化实时监控系统
- 可以保证实时获取数据进行分析
- 支持大屏展示，自适应分辨率
- 支持联动效果，一端操作，多端联动展示

## 技术选型

- HTML、CSS、JS
- ECharts
- Vue2 全家桶
- Axios
- Webpack
- Koa2
- WebSocket

## 启动项目

- 第1步：安装依赖

```ts
yarn
```

- 第2步：启动项目

```
yarn serve
```
